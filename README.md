# 📊 Sales Data Analysis Capstone Project

## 📌 Project Overview  
This capstone project focuses on analyzing retail sales data to uncover key business insights and provide strategic recommendations.  

The analysis was conducted using SQL for querying, Excel for data cleaning, and Power BI for visualization and dashboard creation.

---

## 🛠 Tools & Technologies  
- SQL (Data querying and transformation)  
- Microsoft Excel (Data cleaning and preprocessing)  
- Power BI (Data visualization and dashboard development)  

---

## 📁 Project Structure  
- data/ → Cleaned dataset used for analysis  
- sql/ → SQL queries for data extraction and analysis  
- dashboard/ → Power BI dashboard file (.pbix)  
- reports/ → Insights and recommendations document  

---

## 🔍 Key Insights  

### 🌍 Country Performance  
- The UK is the dominant market, generating the highest revenue and customer base.  
- The Netherlands shows the highest average transaction value despite fewer transactions.  
- Greece and Israel underperform in both revenue and customer activity.  

### 📦 Product Performance  
- Paper Craft Little Birdie is the top-performing product in both revenue and quantity.  
- Several products show extremely low sales, indicating poor market performance.  
- Low-cost items like Popcorn Holder and WW2 Gliders drive high sales volume.  

### 📅 Time-Based Trends  
- September 2019 recorded the highest revenue and activity.  
- Significant growth was observed from 2018 to 2019.  
- Certain months show low engagement, indicating seasonality.  

### 👥 Customer Behavior  
- High-value customers contribute significantly to revenue.  
- Some premium products attract repeat buyers.  
- Low-price items encourage frequent purchases.  

---

## 💡 Business Recommendations  

- Increase transaction value in the UK through bundling and upselling strategies  
- Target high-value markets like the Netherlands with premium offerings  
- Improve marketing efforts in underperforming countries  
- Expand availability of top-performing products globally  
- Focus on high-demand, low-price items to maximize sales volume  
- Review or discontinue poorly performing products  
- Leverage seasonal trends for better campaign planning  
- Introduce loyalty programs for high-value customers
